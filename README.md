# Dashboard de Encomendas

Sistema de gerenciamento de pedidos desenvolvido em Python utilizando Streamlit e SQLite.

## Funcionalidades

- Cadastro de pedidos
- Listagem de pedidos
- Atualização de status (pendente / entregue)
- Exclusão de pedidos

## Tecnologias utilizadas

- Python
- Streamlit
- SQLite
- Pandas

## Como executar

1. Clone o repositório
2. Instale as dependências

pip install -r requirements.txt

3. Execute o projeto

streamlit run app.py
